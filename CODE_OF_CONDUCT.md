a# Code of Conduct

## Our Commitment

We want EnvGuardian to be an open, welcoming, and respectful project for everyone who wants to participate.

Contributors, maintainers, and community members are expected to interact in a professional and constructive way.

## Expected Behavior

Please:

- be respectful
- be constructive
- assume good intent
- welcome feedback
- communicate clearly
- focus on the work, not the person

## Unacceptable Behavior

The following behaviors are not acceptable:

- harassment or discrimination
- personal attacks
- insulting or demeaning language
- trolling or intentionally disruptive behavior
- publishing private information without permission
- repeated bad-faith arguments

## Scope

This Code of Conduct applies to:

- GitHub issues
- pull requests
- discussions
- project documentation spaces
- any community interactions related to EnvGuardian

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that do not align with this Code of Conduct.

Repeated or serious violations may result in temporary or permanent exclusion from the project spaces.

## Reporting

If you experience or witness unacceptable behavior, please report it to the project maintainers through the repository issue tracker or the contact method that will be defined for the project.
